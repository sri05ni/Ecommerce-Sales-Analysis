# E-Commerce Sales Analysis Dashboard

## Project Overview
This project focuses on analyzing e-commerce sales data using Excel, MySQL, and Power BI. The objective of the project is to identify sales trends, profit distribution, customer segment performance, and regional business insights through data analysis and visualization.

---

## Tools & Technologies Used
- Microsoft Excel
- MySQL
- Power BI

---

## Project Workflow
1. Data Cleaning and Preparation using Excel
2. Data Import and Analysis using MySQL
3. Interactive Dashboard Creation using Power BI
4. Business Insight Generation

---

## Key Analysis Performed
- Total Sales Analysis
- Profit Analysis
- Sales by Category
- Regional Profit Analysis
- Monthly Sales Trend Analysis
- Customer Segment Analysis
- Profit Margin Calculation
- Top Performing Products Analysis

---

## Dashboard Features
- KPI Cards for Sales, Profit, Quantity, and Profit Margin
- Interactive Charts and Visualizations
- Regional and Category-wise Analysis
- Monthly Sales Trend Tracking
- Dynamic Filters (Slicers)

---

## Key Insights
- Technology category generated the highest sales.
- West region contributed maximum profit.
- Consumer segment produced the highest revenue.
- Profit margins varied across product categories.
- Monthly sales trends highlighted peak business periods.
